border - Borde

Permite definir reglas para todos los bordes. Con border se pueden definir los siguientes valores: width, border-style, border-color.

border-left, border-right, border-top, border-bottom

Sirve para modificar especificamente los bordes izquierdo, derecho, superior o inferior.

Border radius - radio de borde

Redondea los bordes de un elemento.

background-image

Coloca una imagén de fondo

color

Cambia el color de los elementos de texto, puede definirse en hexadecimal, nombre del color o en "rgb()"

font - fuente

Cambia las propiedades del texto como: tamaño, variante, etc.

font-family

Cambia la fuente del texto, se pueden importar varias desde el documento de html y en el css se especifican

font-style, font-weight

Sirve para cambiar: estilo y el peso ( bold ).

width, height

Modifica las propiedades de anchura y altura de los bloques o de los elementos como imagenes.

list-style - estilo de lista

Sirve cambiar el estilo de una lista
